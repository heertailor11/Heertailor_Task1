# Exploratory Data Analysis (EDA) Project

## Overview

This project performs Exploratory Data Analysis (EDA) on two different datasets:

1. **Retail Sales Dataset**
2. **Food Menu Dataset**

The objective is to understand data patterns, customer behavior, sales trends, and nutritional insights using Python data analysis and visualization libraries.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Dataset 1: Retail Sales EDA

### Objectives

* Analyze customer demographics.
* Understand purchasing behavior.
* Identify top-selling products and categories.
* Examine seasonal sales trends.
* Study the impact of discounts and previous purchases.

### Analysis Performed

#### Data Inspection

* Dataset shape
* Data types
* Missing value detection
* Statistical summary

#### Descriptive Statistics

* Mean
* Median
* Mode
* Standard Deviation

#### Visualizations

* Gender Distribution
* Sales by Category
* Top 10 Purchased Products
* Customer Age Distribution
* Season-wise Sales
* Sales by Payment Method
* Discount vs Amount Analysis
* Previous Purchases vs Amount
* Correlation Heatmap

### Key Insights

* Identifies customer demographics.
* Highlights popular product categories.
* Reveals seasonal purchasing patterns.
* Shows relationships between discounts and sales amounts.
* Helps businesses make data-driven decisions.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/eda-project.git
```

2. Navigate to the project directory:

```bash
cd eda-project
```

3. Install required packages:

```bash
pip install pandas numpy matplotlib seaborn
```

---

## Running the Project

Open the notebook using Jupyter Notebook:

```bash
jupyter notebook Oasis1.ipynb
```

Run all cells sequentially to generate the analysis and visualizations.
